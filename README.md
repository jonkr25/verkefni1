#verkefni 1

1.
cd-change directory sem skiptir leiðina þína
ls -  segir þér hvar þú ert
pwd - segir þér hvaða möppu þú ert að vinna í 
mkdir - gerir nýja möppu í directory sem þú ert í

2.
git clone - sækir verkefnið og býr til möppu með öllum skrám
git status- sýnir hvaða skrár hafa breyst
git diff- Ber saman mun á milli útgáfa skráa

3.
git log- segir eiganda tíma og dagsetningu
git checkout - endurheimtir skrá sem þú vast síðast í
git branch - stjórnar greinum(branches) þegar ég skrifaði það inn kom master út af ég var í master greininni

4. sýnir þér breytingar af skrám og getur gert mikið.
5. Helstu kostir að nota GIT er að það er létt að vinna með einhverjum með að nota git pull og push. 
6. ég tel að gera færslur oft og stutt á milli þeim út af ég bíð of lengi getur commitið verið of oft
7. Working directory er er vinnumappa sem er staður sem þú ert að vinna í og Staging Area er eins og biðstofa fyrir breytingar sem þú vilt setja í næsta commit.
